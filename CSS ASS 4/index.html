CSS Box Model Demo

Content Box Model

This box uses the 'content-box' model.

In the 'content-box' model, the width and height of the element apply to the content only. Padding, border, and margin are added outside of the width and height of the content.

Border Box Model

This box uses the 'border-box' model.

In the 'border-box' model, the width and height of the element include the content, padding, and border. Margin is added outside of the width and height of the border.
